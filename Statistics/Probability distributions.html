Random Variables

Discrete vs Continuous Random Variables

A random variable is a variable whose value is determined by the outcome of a random experiment. It maps outcomes (like flipping a coin) to numbers.

🎯 Discrete RV

Takes countable values (whole numbers, finite steps)

Examples • Number of heads in 10 coin flips (0–10)
• Number of cars passing in an hour
• Dice roll result (1, 2, 3, 4, 5, 6)

〰️ Continuous RV

Takes any value in a range (infinite possibilities)

Examples • Height of a person (170.234... cm)
• Temperature today (34.7°C)
• Time to finish a race (in seconds)
Visualization: Discrete (Bar) vs Continuous (Curve)

📊 Probability Mass Function (PMF) Discrete Only

PMF gives the exact probability that a discrete RV equals a specific value.

P(X = x) = probability that X takes value x

Rules:
1. P(X = x) ≥ 0 for all x
2. ΣP(X = x) = 1 (all probabilities sum to 1)
Example: Fair Coin (2 flips) X = number of heads. Possible values: 0, 1, 2
P(X=0) = 1/4 = 0.25 (TT)
P(X=1) = 2/4 = 0.50 (HT or TH)
P(X=2) = 1/4 = 0.25 (HH)
Sum = 1.0 ✓

📈 Probability Density Function (PDF) Continuous Only

For continuous RVs, we cannot assign probability to exact values (P(X = 170.00000) = 0). Instead, PDF gives probability over an interval.

P(a ≤ X ≤ b) = ∫[a to b] f(x) dx

Rules:
1. f(x) ≥ 0 for all x
2. ∫[-∞ to ∞] f(x) dx = 1 (total area under curve = 1)
Example: Heights What's the probability that a random person is between 165–175 cm?
→ P(165 ≤ X ≤ 175) = area under the PDF curve from 165 to 175
You cannot ask: "What's P(X = 170)?" — it equals 0!
PDF: Shaded Area = Probability in that Interval

📉 Cumulative Distribution Function (CDF)

CDF answers: "What is the probability that X ≤ x?" — works for both discrete and continuous RVs.

F(x) = P(X ≤ x)

For discrete: F(x) = Σ P(X = k) for all k ≤ x
For continuous: F(x) = ∫[-∞ to x] f(t) dt

CDF always goes from 0 → 1, never decreases
Example: Dice Roll F(3) = P(X ≤ 3) = P(1) + P(2) + P(3) = 1/6 + 1/6 + 1/6 = 0.5
F(6) = 1.0 (probability of getting 6 or less = 100%)

Practical Use: "What % of students scored below 80?" → CDF!
CDF of a Dice Roll (Step Function)
PMF/PDF tells you probability AT a point. CDF tells you probability UP TO a point. Both are essential and complement each other.

Discrete Distributions

Bernoulli Distribution Discrete

The simplest distribution — a single trial with two outcomes: Success (1) or Failure (0).

Parameters: p (success prob) Mean: p Variance: p(1-p)
P(X = 1) = p ← success
P(X = 0) = 1 - p ← failure

X ∈ {0, 1} — only two possible values!
Real-World Example A patient takes a medicine. It either works (X=1) or doesn't (X=0).
If medicine works 70% of time: p = 0.7
P(works) = 0.7, P(doesn't work) = 0.3

Other examples: Spam/not-spam email, Win/lose a game, Click/no-click on an ad
When to Use Any binary outcome experiment with one trial. It's the building block for Binomial distribution!
Bernoulli PMF (p = 0.7)

Binomial Distribution Discrete

Extension of Bernoulli — n independent trials, each with success probability p. X = total successes.

Parameters: n, p Mean: np Variance: np(1-p) X ∈ {0, 1, ..., n}
P(X = k) = C(n,k) · p^k · (1-p)^(n-k)

where C(n,k) = n! / (k! · (n-k)!) ← "n choose k"

Must satisfy: fixed n, independent trials, same p each time
Example: Product Defects A factory produces 20 items. Each has a 10% chance of being defective (p=0.1, n=20).
Q: Probability of exactly 3 defective items?
P(X=3) = C(20,3) · 0.1³ · 0.9¹⁷
= 1140 · 0.001 · 0.167 ≈ 0.190 (19%)

Expected defectives = np = 20 × 0.1 = 2 defectives
When to Use Fixed number of trials, binary outcomes, constant probability, independent trials. Examples: quality control, medical trials, election polling, A/B testing
Binomial PMF: n=10, p=0.3 vs p=0.5 vs p=0.7

Poisson Distribution Discrete

Models the number of events occurring in a fixed time interval when events happen at a constant average rate, independently.

Parameter: λ (lambda) = average rate Mean = λ Variance = λ X ∈ {0, 1, 2, ...}
P(X = k) = (e^(-λ) · λ^k) / k!

where e ≈ 2.718, λ = average events per interval

Key: Mean = Variance = λ (unique property!)
Example: Call Center A customer service center receives an average of 5 calls per hour (λ=5).
Q: Probability of receiving exactly 3 calls in the next hour?
P(X=3) = (e^(-5) · 5³) / 3!
= (0.0067 × 125) / 6 ≈ 0.140 (14%)

Other examples: Website traffic per minute, emails per day, earthquakes per year, typos per page
When to Use Events are rare & random in a continuous interval, events are independent, constant average rate. Rule of thumb: n is large, p is small, and np = λ is moderate → use Poisson.
Poisson PMF: Different λ Values

Geometric Distribution Discrete

Models the number of trials needed until the first success. "How long do we have to wait?"

Parameter: p Mean: 1/p Variance: (1-p)/p² X ∈ {1, 2, 3, ...}
P(X = k) = (1-p)^(k-1) · p

k = number of trials until first success
(1-p)^(k-1) = k-1 failures before success
Example: Free Throw in Basketball A player makes 60% of free throws (p=0.6).
Q: Probability they need exactly 3 attempts to make one?
P(X=3) = (0.4)² × 0.6 = 0.16 × 0.6 = 0.096 (9.6%)
Expected throws needed = 1/p = 1/0.6 ≈ 1.67 throws

Other examples: Calls needed to close a sale, server retries, job interviews to get offer
When to Use You're counting trials until first success. Memoryless property: past failures don't affect future probability.

🗂️ Quick Comparison: Which Discrete Distribution?

Distribution Question Asked Key Clue Example
Bernoulli Success or Failure? Single trial, binary outcome Will this email be spam?
Binomial How many successes in n trials? Fixed n, same p How many of 20 products are defective?
Poisson How many events in time/space? Rate λ, no fixed n How many calls per hour?
Geometric How many trials until first success? Waiting for first event How many tries to get a job?

Continuous Distributions

Uniform Distribution Continuous

Every value in the range [a, b] has equal probability. The most basic continuous distribution.

Parameters: a (min), b (max) Mean: (a+b)/2 Variance: (b-a)²/12
f(x) = 1/(b-a) for a ≤ x ≤ b
f(x) = 0, otherwise

PDF is a flat horizontal line — all values equally likely!
Example: Bus Arrival A bus arrives every 10 minutes. You arrive at a random time. Wait time X ~ Uniform(0, 10).
P(wait between 3 and 7 min) = (7-3)/(10-0) = 4/10 = 40%
Expected wait = (0+10)/2 = 5 minutes

Other examples: Random number generators, rounding errors, arrival times
Uniform Distribution PDF — Flat Line
When to Use When all outcomes are equally likely in a range. Used in random number generation, simulation, and as a default "no information" prior in Bayesian stats.

⭐ Normal (Gaussian) Distribution Most Important!

The king of distributions. Bell-shaped, symmetric, appears everywhere in nature. The Central Limit Theorem ensures that averages of many random variables tend to be normally distributed.

Parameters: μ (mean), σ (std dev) Notation: X ~ N(μ, σ²) Symmetric around μ Mean = Median = Mode
f(x) = (1 / (σ√(2π))) · exp(-(x-μ)² / (2σ²))

The famous bell curve!
Example: Exam Scores Class exam scores: μ = 70, σ = 10. X ~ N(70, 100).
• 68% of students score between 60–80 (μ ± σ)
• 95% of students score between 50–90 (μ ± 2σ)
• 99.7% score between 40–100 (μ ± 3σ)

Other examples: Heights, weights, IQ scores, measurement errors, stock returns (approximately)
Normal Distribution — The 68-95-99.7 Rule (Empirical Rule)
The 68-95-99.7 rule is crucial for data science! 95% of data lies within 2σ of the mean, and values beyond 3σ are considered extreme outliers.
When to Use Heights, weights, IQ, blood pressure, measurement errors, exam scores. Also use when Central Limit Theorem applies (sample means of large datasets). Most real-world data is approximately normal!

Standard Normal Distribution & Z-Scores Continuous

Special case of Normal where μ=0, σ=1. Used to standardize any normal distribution for comparison.

Z = (X - μ) / σ ← Z-score formula

Z tells you: "How many standard deviations is X from the mean?"

Z ~ N(0, 1)
f(z) = (1/√(2π)) · e^(-z²/2)
Example: Comparing Scores Alice: Physics score 75/100 in a class with μ=65, σ=10
Bob: Math score 85/100 in a class with μ=80, σ=15

Alice's Z = (75-65)/10 = +1.0 (1 SD above mean)
Bob's Z = (85-80)/15 = +0.33 (0.33 SD above mean)

→ Alice performed better relative to her class!

Z-score lookup: Z=1.0 → 84th percentile (84% scored below Alice)
Standard Normal Z-Distribution (μ=0, σ=1)
Memorize these Z-score percentiles: Z=0→50%, Z=1→84%, Z=1.65→95%, Z=1.96→97.5%, Z=2.33→99%, Z=2.58→99.5%

Exponential Distribution Continuous

Models the time between events in a Poisson process. The continuous version of Geometric distribution.

Parameter: λ (rate) Mean: 1/λ Variance: 1/λ² Memoryless property!
f(x) = λ · e^(-λx) for x ≥ 0

F(x) = 1 - e^(-λx) ← CDF

Memoryless: P(X > s+t | X > s) = P(X > t)
Example: Server Failure A server crashes on average once every 100 hours. λ = 1/100 = 0.01.
Q: Probability it runs more than 200 hours without crashing?
P(X > 200) = e^(-0.01×200) = e^(-2) ≈ 13.5%

Memoryless: Even if server has already run 100 hours safely, probability of running 200 MORE hours is still e^(-2) = 13.5%!

Other examples: Call center wait time, time between arrivals, component lifespan
Exponential PDF for Different Rates (λ)
When to Use Modeling waiting times, time-to-failure, inter-arrival times in queues. If events happen at a constant rate (Poisson), the time BETWEEN events is Exponential.

Log-Normal Distribution Continuous

If ln(X) is normally distributed, then X is log-normally distributed. Always positive, right-skewed.

If Y ~ N(μ, σ²), then X = e^Y ~ LogNormal(μ, σ²)

Mean = e^(μ + σ²/2)
Variance = (e^σ² - 1) · e^(2μ + σ²)
Example: Stock Prices & Income Stock prices can't go negative and have multiplicative changes (% returns). Stock returns are approximately normal, so stock prices are log-normal.

City income distribution: Few billionaires, many middle-income people → right-skewed → Log-normal

Other examples: City populations, biological concentrations (bacteria, virus load), insurance claim sizes
When to Use Data is positive and right-skewed. Growth processes (compound interest, population growth). When taking the log of your data makes it normally distributed, use Log-Normal.

Distribution Shape Key Use Case Real Example

Special Distributions

t-Distribution (Student's t) Special

Looks like a normal distribution but with heavier tails. Used when the population standard deviation is unknown and sample size is small.

Parameter: ν (degrees of freedom = n-1) Mean: 0 As ν→∞, t → Normal
t = (X̄ - μ) / (s / √n)

where s = sample std dev, X̄ = sample mean, μ = true mean

Use Z when σ known, use t when σ unknown (almost always!)
Example: Drug Testing You test a new drug on 15 patients. You measure blood pressure reduction and want to test if it's significantly different from zero. Population σ is unknown.
You calculate t = 2.14 with df = 14.
From t-table: critical value at α=0.05 (two-tail) = 2.145
Since 2.14 < 2.145 → Fail to reject H₀ (barely not significant)

Other uses: Any hypothesis test with small sample and unknown σ, confidence intervals for means
t-Distribution: Heavier Tails than Normal (more uncertainty with small n)
When to Use t-Distribution Small sample size (n < 30), population standard deviation unknown, data approximately normal. Rule: When in doubt, use t (it becomes z when n is large anyway). Used in t-tests (one-sample, two-sample, paired).

Chi-Square Distribution (χ²) Special

Sum of k squared standard normal variables. Skewed right, always positive. Used for testing variance and categorical data.

Parameter: k (degrees of freedom) Mean: k Variance: 2k Always ≥ 0
χ² = Z₁² + Z₂² + ... + Zₖ²

where Zᵢ ~ N(0,1) independently

Chi-square test statistic: χ² = Σ (Observed - Expected)² / Expected
Example: Goodness of Fit Test A dice is rolled 120 times. Expected: 20 times each face. Observed: 25, 17, 18, 22, 20, 18.
χ² = (25-20)²/20 + (17-20)²/20 + ... = 25/20 + 9/20 + 4/20 + 4/20 + 0 + 4/20
χ² = 1.25 + 0.45 + 0.2 + 0.2 + 0 + 0.2 = 2.3
Critical value (df=5, α=0.05) = 11.07. Since 2.3 < 11.07 → Dice is fair ✓

Other uses: Independence tests in contingency tables, testing if data fits a distribution
Chi-Square PDF: Right-skewed, shifts right as df increases
When to Use Chi-Square Goodness-of-fit tests (does data follow expected distribution?), independence tests in contingency tables (are two categorical variables related?), testing population variance.

F-Distribution Special

Ratio of two chi-square distributions (scaled). Used to compare variances of two or more groups. Appears in ANOVA.

Parameters: d₁, d₂ (two df) Always ≥ 0 Right-skewed
F = (χ²₁/d₁) / (χ²₂/d₂) = ratio of two variances

ANOVA F-statistic: F = Between-group variance / Within-group variance

F > 1 means more variation between groups than within
Example: ANOVA — Comparing Teaching Methods 3 teaching methods tested on 4 groups each. Are the methods different in effectiveness?
Between-group variance = 120, Within-group variance = 40
F = 120/40 = 3.0
Critical F(2, 9) at α=0.05 = 4.26
Since 3.0 < 4.26 → No significant difference between methods

Other uses: Comparing variability of two processes, regression model significance testing
When to Use F-Distribution ANOVA (comparing means of 3+ groups), testing if regression model is significant, comparing two population variances. ANOVA is F-test's most famous application.

Beta Distribution Special

Very flexible distribution on [0, 1]. Perfect for modeling probabilities and proportions. A foundational tool in Bayesian statistics.

Parameters: α (alpha), β (beta) X ∈ [0, 1] Mean: α/(α+β) Very flexible shape!
f(x) = x^(α-1) · (1-x)^(β-1) / B(α,β)

where B(α,β) = Γ(α)Γ(β)/Γ(α+β) is the normalizing constant

α=β=1 → Uniform. α=β>1 → Symmetric bell. α≠β → Skewed
Example: A/B Testing (Bayesian) You're testing two website buttons. Button A had 30 clicks from 100 visitors. What's the true click rate?

Prior belief: Beta(1,1) (no prior knowledge = uniform)
After observing: 30 clicks, 70 non-clicks
Posterior: Beta(1+30, 1+70) = Beta(31, 71)
Most likely click rate = (α-1)/(α+β-2) = 30/100 = 30%
95% credible interval using Beta distribution: approximately 21%–40%

Other uses: Modeling success rates, conversion rates, disease prevalence, proportions
Beta Distribution: Different α and β → Different Shapes
When to Use Beta Distribution Modeling probabilities, proportions, rates (anything bounded between 0 and 1). Gold standard in Bayesian A/B testing. Prior distribution for binomial parameter p. Machine learning model calibration.

📊 Master Comparison: All Special Distributions

Distribution Built From Primary Use Key Test
t-Distribution Normal, small n Comparing means when σ unknown t-test (one, two sample, paired)
Chi-Square (χ²) Sum of Z² Categorical data, variance testing Goodness-of-fit, independence test
F-Distribution Ratio of χ² Comparing multiple group means ANOVA, regression F-test
Beta Proportions [0,1] Modeling probabilities/rates Bayesian inference, A/B testing
Quick Decision Rule: Testing means? → t-test (uses t-dist). Testing proportions/frequencies? → Chi-square. Comparing 3+ groups? → ANOVA (uses F-dist). Bayesian modeling of probabilities? → Beta.
